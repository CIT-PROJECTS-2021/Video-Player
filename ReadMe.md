# Python Video Player

## Library Used

- PyQt

## Project Setup

Follow the steps below by typing the commands as shown to step up and run the application

Clone project repository to your computer

```
https://github.com/CIT-PROJECTS-2021/
Video-Player.git
```

Change directory to `videoPlayer`

```
cd videoPlayer
```

Create and activate virtual environment

```
pip install virtualenv
```

```
virtualenv venv
```

On MacOs/Ubuntu Platform

```
venv/bin/activate
```

On Windows Platform

```
vevn/Scripts/activate.bat
```

## Installing Library

```
pip install PyQt5
```
